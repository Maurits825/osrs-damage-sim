Load from Runelite Setup Guide
1. HTTP API Plugin

Get the HTTP API plugin from the plugin hub. Make sure the plugin is on.

2. Bypass CORS

Some program has to be used to bypass CORS and access the api of the plugin.
Requestly can be used for this.
Create a new Modify Headers rule with the following settings:

  • URL contains http://localhost:8080/equip
  • Response Headers
    • Header: Access-Control-Allow-Origin
    • Value: https://maurits825.github.io

(Optional) If running locally, duplicate the above rule and set the value to http://localhost:4200.

More info on these CORS settings.

3. Load gear setup from RL

The Load from Runelite button should now load the items you have equipped in Runelite.
