1. HTTP API Plugin

Get the HTTP API plugin from the plugin hub. Make sure the plugin is on.

2. Bypass CORS

Some program has to be used to bypass CORS and access the api of the plugin.
Requestly can be used for this.
Create a new Modify Headers rule with the following settings:

  • URL contains http://localhost:8080/equip
  • Response Headers
      Production Environment
    • Header: Access-Control-Allow-Origin
    • Value: https://maurits825.github.io/osrs-damage-sim
      • Local Environment (Optional)
    • Header: Access-Control-Allow-Origin
    • Value: http://localhost:4200
